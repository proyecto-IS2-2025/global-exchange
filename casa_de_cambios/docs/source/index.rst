Global Exchange Documentation
==============================

Bienvenido a la documentación de Global Exchange, un sistema de gestión de casa de cambios desarrollado con Django.

Esta documentación incluye información detallada sobre los módulos, modelos y funcionalidades del sistema.

Módulos Principales
====================

El sistema está organizado en los siguientes módulos principales:

* **Clientes**: Gestión de información de clientes, segmentos y asignaciones
* **Usuarios**: Sistema de autenticación y perfiles de usuario  
* **Transacciones**: Procesamiento de operaciones de cambio de moneda
